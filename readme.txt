== Description ==

**紫茉莉-基础框架，紫茉莉系列产品用于后台管理的基础框架。**

紫茉莉系列主题、插件都采用此基础框架生成后台管理页面。不用担心使用此基础框架会造成系统资源浪费，采用此种方式能够减少主题或插件文件大小，并且在安装多个插件时能够减少文件的重复加载，减少资源占用。

框架支持快速生成后台设置页面、TERM 选项设置、文章 Metabox 内容。支持常用各类表单，增加了 Ajax、标签输入、标签选择、文件上传、颜色选择、文本选择、图像选择等表单功能，图片相关表单可实现预览功能。简化了表单生成、数据保存，同时还提供了选项数据快速备份导出和恢复的功能。

== Installation ==

= 插件需求 =

* PHP 7.3 及以后版本。
* WordPress 5.6 及以后版本。
* 推荐使用最新的 PHP 和 Wordpress 版本。

= 插件设置 =

* 本插件无选项设置。
* 插件介绍位于 “插件” 菜单下 “紫茉莉 - 基础框架” 页面。

== Frequently Asked Questions ==

= 必须安装基础框架吗 =

如果不安装 “基础框架”，紫茉莉系列主题或插件无法进入参数设置页面。

= 完成设置后能删除吗 =

完成参数设置后，可以停用或删除 “基础框架”，不影响紫茉莉系列主题或插件运行。
